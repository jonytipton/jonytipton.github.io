Unity VR Game

Escape The Grid

Utilizing C# and Unity XR-frameworks, I created an immersive VR game inspired by Disney's TRON universe - specifically the arena disc battles. This project involved extensive troubleshooting of VR hardware and software, as well as needing to continually adapt to new changes of pre-release development packages.

UI/UX testing was conducted throughout the development process and user feedback informed prioritization of planned gameplay and graphics improvements.

All graphics and gameplay code were created by myself. This required independently learning 3D modeling, texturing, and animation in AutoDesk Maya and XR packages by Oculus + Unity.

Audio Wayfinding App

Theia Wayfinding

I partnered with a local accessibility group to identify problem areas in the daily lives of people with disablities and how technology can alleviate them. Enabling independent wayfinding for those with visual impairments was one of the challenges shared - and a unique application for augmented reality.

Developed with Microsoft's Azure Spatial Anchors SDK and Unity, I created an audio based wayfinding app that uses AR and the cloud to reference location-based sound queues that inform users of their heading and surroundings. Each anchor has an assigned audio clip and positioning data that considers GeoLocation (GPS), WiFi signals, Bluetooh (LE beacons), and visual markers.

Cross-Platform App

Ecological Momentary Assessment

Notification alert app designed to facilitate psychological research studies. "In the moment" behavioral survey assessments are sent to participant devices via push notification - utilizing Firebase Messaging and cloud SDKs.

The Flutter cross-platform framework allows for app deployment to Android and iOS. Administrative backend implemented with Python (Django) allows for easy management and scheduling of surveys during a study.
